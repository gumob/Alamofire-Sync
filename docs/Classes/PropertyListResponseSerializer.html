PropertyListResponseSerializer

public final class PropertyListResponseSerializer : ResponseSerializer

A ResponseSerializer that performs minimal response checking and returns any response PropertyList as-is. By default, a request returning nil or no data is considered an error. However, if the request has an HTTPMethod or the response has an HTTP status code valid for empty responses, then an empty Data value is returned.
